Art Gallery

Art Gallery is a Flask-based web application that allows users to explore famous artworks and share their own creations with the community.

## ✨ Features

- Browse a collection of artworks
- View detailed artwork information
- Search artworks by title or artist
- Filter artworks by category
- User registration and login system
- Upload your own artworks
- Save artworks to favorites
- View your uploaded artworks
- Pagination for easier navigation
- Responsive and modern design

## 🛠 Technologies Used

- Python
- Flask
- Flask-SQLAlchemy
- Flask-Login
- SQLite
- HTML
- CSS
- Bootstrap
